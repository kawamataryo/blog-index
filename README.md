# Blog Index（wip）

An app to aggregate index a tech blog with cloud functions.

## Feature

### Create blog index API

Get index the following service.

- [Qiita](https://qiita.com/)
- [note](https://note.com/)
- [Zenn](https://zenn.dev/)
- [Twitter](https://developer.twitter.com/en)

### Create slack Bot 

Post your blog index to slack.
